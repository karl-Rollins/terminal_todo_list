# 🏷 Project Name

TODO_LIST

A bash script that permits users to add, display and delete thier to do tasks in the terminal.

## 📌 Problem Statement

Developers forget what they are to do. A quick way to get from the terminal a displayed list will help.

## 🎯 Project Goals

Allows user to do task such as;

- Add tasks to a list
- Save list to a .txt file
- See the list
- Delete tasks from list

## 🛠 Tech Stack

Bash

## ⚙ Installation & Setup

Open the folder todo_list in your IDE(eg vscode)
All code is written in the todo.sh file.

In the terminal;
-run the code chmod +x todo.sh to give permission to the file to be executed
-then run the code ./todo.sh.
-Add argument(add,list, clear,del) eg "./todo.sh add" to do the task your require
